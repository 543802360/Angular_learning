# NgZorroCesium

Angular搭建Cesium开发平台，使用ng-zorro组件可和primeng组件库对Cesium进行上层的封装。目前构建了四个组件和两个服务  

---

## 组件

- layout组件
- viewer组件
- scenesetting组件
- layertree组件
- overlayerTree组件
- mouseposition组件

## 服务

- cesiumservice
- viewerfactory

### 使用

安装依赖
```
npm install
## or 
cnpm install

```

debug

```
npm run debug
```

release

```
npm run release
```

在浏览器中访问 [http://127.0.0.1:10010](http://127.0.0.1:10010)
